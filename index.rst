欢迎来到 Flask
================

.. image:: _static/logo-full.png
   :alt: Flask: web development, one drop at a time
   :class: floatingflask

欢迎来到 Flask 文档。  本文档分为几个部分。我推荐你从 :ref:`installation` 开始，然后再前往 :ref:`quickstart` 章节。除快速入门外， :ref:`tutorial`  更详细的介绍了怎样用 Flask 创建一个完整（虽然小）的应用。如果你想深入了解 Flask 内部，查阅 :ref:`api` 文档。 常见的模式将在 :ref:`patterns` 章节叙述。

Flask 依赖于两个外部库： `Jinja2`_ 模板引擎和 `Werkzeug`_ WSGI 工具包。这些库的文档不包含在此文档中。如果你想深入了解，请查看以下链接：

-   `Jinja2 文档 <http://jinja.pocoo.org/2/documentation/>`_
-   `Werkzeug 文档 <http://werkzeug.pocoo.org/documentation/>`_

.. _Jinja2: http://jinja.pocoo.org/2/
.. _Werkzeug: http://werkzeug.pocoo.org/
